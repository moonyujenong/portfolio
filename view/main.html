SERVICES

  • Responsive

    웹과 모바일에 모두 맞춘 반응형 페이지를 제작하여 페이지에 접근하는 다양한 디바이스에 대응하며, 뿐만아니라 PC/Mobile 각 각에 맞춘 적응형 웹 또한 제작 가능합니다.

  • GitLab

    작업한 파일을 GitLab 에 pull, push 하여 작업물을 안전하게 백업하고, 팀원들과 공유하여 쉬운 작업이 가능합니다.

  • Front-end

    주로 HTML5, CSS3, SCSS 를 사용하여 퍼블리싱 작업을 진행하며, 필요에 따라 Javascript, jquery 도 사용하여 간단한 이벤트 동작이나 스크립트 구현이 가능합니다. 또한 지도같은 경우 API 를 이용하여 구현 가능합니다.

  • Library

    배너나 나열된 객체들의 슬라이드와 같은 이벤트 동작이 필요할 경우 적절한 플러그인 및 프레임 워크 등을 찾아 코드에 대입하여 구현 가능합니다.

PORTFOLIO

구인구직 사이트
(반응형)

해당 사이트는 구인구직 하는 이용자들을 위해 사용되고 있으며 구직자들뿐만 아니라 구인하는 업주들도 공고를 낼 수 있습니다. 지역/직업 찾기가 모달 팝업으로 구성되어 있으며 메인 및 서브의 상단 배너는 Library를 통해 작업되었습니다.

자격증 사이트
(반응형)

해당 사이트는 자격증에 대한 정보를 모두 모아놓은 사이트 입니다. 기존 사이트에서 리뉴얼 된 디자인으로 퍼블리싱 되었습니다. 메인 및 서브 페이지에서의 slideToggle 기능은 Jquery & javascript 로 제작되었으며 메인 와이드 배너는 swiper 기능을 이용해 제작하였습니다.

기업 홈페이지
(반응형)

해당 사이트는 데이터 및 서버 구축 용도의 사이트로 기존 사이트에서 리뉴얼 된 디자인으로 퍼블리싱 되었습니다. 해당 사이트의 slide 기능 또한 Jquery 로 제작하였습니다.

기업 홈페이지2
(반응형)

해당 사이트는 기업에서 운영하는 홈페이지로 기업의 공정 및 장비에 대한 과정&설명이 포함되어 있습니다. 회사소개 > 사업장 현황의 지도 Api 같은 경우, 구글지도 및 카카오 지도 API 를 사용하여 제작하였습니다.

개인 홈페이지 제작 의뢰
(반응형)

해당 사이트는 오프라인 마켓을 온라인 시장까지도 알리기 위한 페이지로, 외주를 받아 작업하였습니다. background image 가 고정되어 있어 정적인 웹 사이트에 생동감을 느끼게 해줍니다. 사이트 작업 과정에서 간단한 애니메이션 스크립트가 사용되었습니다.
